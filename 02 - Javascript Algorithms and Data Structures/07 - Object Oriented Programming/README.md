 ## 7.1 JavaScript Objects

### 7.1.1 Creating Objects With Methods and Properties

 * We can easily declare an object with properties and methods using {} braces

 * E.g.

        let Dog = { name: "Scooby",
                    legs: "4",
                    sayName: function(){
                        console.log("My name is "+Dog.name)
                    }}
