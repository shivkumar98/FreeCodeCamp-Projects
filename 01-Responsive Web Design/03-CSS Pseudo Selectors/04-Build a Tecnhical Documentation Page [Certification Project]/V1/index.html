Encapsulation

Encapsulation is a key design principle from object-oriented design

Conceptually, it means classes should wrap data and methods which operate on its data within the class itself

This is achieved by using access modifiers and making fields private, and getters/setters public

class Anime { private String title; private Genre genre; // public setters and getters }

Encapsulation achieves the following:

  • Data-integrity
  • Security
Arrays

Arrays are a funamental data structure in Java

Arrays are ordered and have a fixed size

Arrays is also an interface, it can be used to store primitives and objects

int[] arr = new int[3]

Arrays have the following properties:

  • They have fast read and write speeds
  • You can get the size of an array using .lenght() method
  • Each item of the array has an index which is required to retrieve the item
